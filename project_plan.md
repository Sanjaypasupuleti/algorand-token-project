 Idea

Build a blockchain-based crowdfunding platform where users can create fundraising campaigns and others can donate using Algorand.

 How It Works

User creates a funding campaign (goal amount + deadline).

Contributors send ALGO to the smart contract.

If goal is reached → funds go to creator.

If goal is not reached → contributors get refunds automatically.

 Why This Is Good

Uses smart contract logic

Real-world application

Shows understanding of escrow contracts
